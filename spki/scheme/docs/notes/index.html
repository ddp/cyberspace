Viewing tips by platform

For pixel-perfect box diagrams, use PostScript or PDF instead of HTML.

macOS

open memo-0002-architecture.ps    # TeXShop or Preview
open memo-0002-architecture.pdf   # Preview

Linux / BSD

gv memo-0002-architecture.ps      # Ghostview
evince memo-0002-architecture.pdf # GNOME
zathura memo-0002-architecture.pdf
gs -dNOPAUSE -dBATCH memo-0002-architecture.ps  # raw

Windows

start memo-0002-architecture.pdf  # Default PDF viewer
:: PostScript: install Ghostscript, then gsview

Vault Operations

Core Commands

  • seal-commit — Create sealed version with optional metadata
  • seal-update — Pull latest changes (like svn update)
  • seal-undo — Revert changes (file/hard/soft)
  • seal-history — View commit log with graph
  • seal-branch — Create and switch to branch
  • seal-merge — Merge branches with strategy

Version Management

  • seal-release — Create semantic version (X.Y.Z) with SPKI signature
  • seal-verify — Verify cryptographic seal on release
  • seal-archive — Create archive (tarball/bundle/cryptographic)
  • seal-restore — Restore from archived version
  • seal-migrate — Execute migration between versions
  • seal-check — Verify vault integrity

Archive Formats

  • seal-archive-tarball — Create tar.gz archive
  • seal-archive-bundle — Create git bundle
  • seal-archive-cryptographic — Create encrypted + signed archive
  • seal-restore-cryptographic — Restore cryptographic archive

Configuration

  • vault-init — Initialize vault with signing key
  • vault-config — Get/set vault configuration

Audit Trail

Core Operations

  • audit-init — Initialize audit trail
  • audit-append — Append entry to trail
  • audit-verify — Verify single entry seal
  • audit-chain — Verify entire chain
  • audit-read — Read specific entry

Query Interface

  • audit-by-actor — Find entries by actor principal
  • audit-by-action — Find entries by action verb
  • audit-by-timerange — Find entries in time range

Export Formats

  • audit-export-sexp — Export as S-expressions
  • audit-export-human — Export human-readable text

Entry Structure

  • <audit-entry> — Audit entry record type
  • <audit-actor> — Actor record (principal + authz chain)
  • <audit-action> — Action record (verb + object + params)
  • <audit-context> — Context record (motivation + relates-to)
  • <audit-seal> — Seal record (hash + signature)

Cryptographic Primitives

Ed25519 Operations

  • ed25519-keypair — Generate Ed25519 keypair
  • ed25519-sign — Create Ed25519 signature
  • ed25519-verify — Verify Ed25519 signature
  • save-keypair — Save keypair to files
  • load-keypair — Load keypair from files

Hashing

  • sha512-hash — Compute SHA-512 hash
  • blob->hex — Convert blob to hex string
  • hex->blob — Convert hex string to blob

SPKI Certificates

  • make-spki-cert — Create SPKI certificate
  • verify-spki-cert — Verify certificate
  • parse-spki-cert — Parse certificate S-expression
  • cert->sexp — Convert certificate to S-expression

Certificate Structure

  • <spki-cert> — Certificate record type
  • make-signature — Create signature record
  • hash-alg->string — Convert hash algorithm to string

Utility Functions

Metadata Helpers

  • save-commit-metadata — Save commit metadata to file
  • get-environment-snapshot — Capture environment state
  • get-dependencies-snapshot — Capture dependencies
  • get-git-state-snapshot — Capture git state

Migration Helpers

  • create-migration-marker — Create migration template
  • get-sealed-releases — List all sealed releases

S-Expression Helpers

  • entry->sexp — Convert entry to S-expression
  • sexp->entry — Parse entry from S-expression
  • actor->sexp — Convert actor to S-expression
  • action->sexp — Convert action to S-expression
  • seal->sexp — Convert seal to S-expression

Constants and Variables

  • *vault-config* — Vault configuration alist
  • *audit-config* — Audit configuration alist

## About the HyperSpec

The Common Lisp HyperSpec is a hypertext version of the ANSI Common Lisp standard (ANSI X3.226). It is the authoritative reference for the Common Lisp programming language.

**Primary URL:** https://www.lispworks.com/documentation/HyperSpec/Front/
